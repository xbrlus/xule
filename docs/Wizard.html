Anyone can use a free XBRL US Web Account and API credentials to get up-to-the-minute data from financial reports filed with the US SEC.

XBRL API endpoints:

Fact - filed facts and related metadata

Entity - information about entities submitting reports

Label - human-readable concept equivalents with additional metadata

Report - details about submitted reports

Network - structural details about filed reports (Presentation, Calculation and Definition characteristics)

Relationship - parent-child associations and other details for elements in a discoverable taxonomy set (DTS)

Assertion - issues related to facts in an entity's report (eg. signage, decimals) affecting the overall quality of the information submitted

See "Online Documentation" linked in the Ribbon for full details on endpoints, as well as filter, sort, limit and offset options for the XBRL API.

The =XBRL.showData() custom function can be created or modified directly in the sheet, and accepts valid queries for all endpoints. This custom function accepts three optional parameters to display facets of the information queried. After completing a query, find out about these options from Excel's insert function button under the formulas menu.
